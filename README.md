
# React Native Auto Play

This repository contains the `react-native-autoplay` library, a solution for integrating your React Native application with both **Apple CarPlay** and **Android Auto**.

-   **`react-native-autoplay`**: The core library. For detailed documentation, see the [package README](./packages/react-native-autoplay/README.md).
-   **`example`**: An example application demonstrating the usage of the library. For instructions on how to run the example, see the [example README](./apps/example/README.md).
